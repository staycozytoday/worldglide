## typography

body has `text-transform: lowercase` globally. all text renders lowercase automatically.

### scale

| token | size | weight | usage |
|-------|------|--------|-------|
| page title | `text-[32px] md:text-[40px]` | `font-medium` | h1 on every page |
| section heading | `text-[14px]` | `font-medium` | h2 in admin sections |
| body / card title | `text-[13px]` | `font-medium` | job titles, submission titles |
| body text | `text-[12px]` | normal | descriptions, company names, paragraphs |
| label | `text-[11px]` | normal | form labels, button text, mono metadata |
| caption | `text-[10px]` | normal | meta info, stats, footer text |

### fonts

- **sans**: `Inter, system-ui, -apple-system, sans-serif` â†’ headings, body
- **mono**: `SF Mono, JetBrains Mono, Fira Code, monospace` â†’ labels, stats, buttons, metadata

## conventions

- **all lowercase** â†’ global `text-transform: lowercase` on body
- **separator**: use `ď˝Ą` (middle dot) not `Â·` or `|` or `-`
- **conjunctions**: use `&` not "and"
- **language**: spell/magic themed for grimoire admin ("cast a scrape spell", "casting", "portal spell", "spellbook")
- **no rounded corners** on buttons â†’ flat industrial look
- **no shadows** anywhere
- **no color accents** â†’ monochrome only (exception: `text-red-500` for errors)
- **scrollbar**: 4px thin, minimal, no radius
